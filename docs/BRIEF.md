# 新能源汽车租赁比价平台 - 意图理解（Brief）

## 1. 项目背景

随着新能源汽车的普及，越来越多的用户选择租赁新能源车出行。然而，租车市场平台众多（一嗨、神州、悟空、携程等），价格差异大，用户需要花费大量时间在多个平台之间对比价格，效率低下。

---

## 2. 用户痛点

| 痛点 | 描述 |
|------|------|
| **价格不透明** | 不同平台价格差异大，难以快速找到最优价格 |
| **对比耗时** | 需要打开多个App/网站逐一查询对比 |
| **信息分散** | 车型、续航、座位等信息分散在各平台 |
| **价格波动** | 租车价格随时间变化，错过优惠时机 |

---

## 3. 目标用户

| 用户类型 | 特征 | 核心需求 |
|----------|------|----------|
| **商务出行者** | 频繁租车，对价格敏感 | 快速找到性价比最高的车型 |
| **旅游用户** | 偶尔租车，对流程不熟悉 | 一站式比价，省时省力 |
| **新能源车主** | 体验不同车型 | 查找特定品牌/车型 |

---

## 4. 产品目标

**一句话描述**：新能源汽车租赁一站式比价平台，帮助用户快速找到最优惠的租车方案。

**核心价值**：
- ⏱️ **省时**：一键对比多个平台，无需逐一查询
- 💰 **省钱**：找到最优价格，降低租车成本
- 📊 **透明**：价格历史记录，辅助决策

---

## 5. 功能范围

| 功能模块 | 描述 | 优先级 |
|----------|------|--------|
| **车型搜索** | 支持车型、城市、时间搜索 | P0 |
| **多平台链接** | 一键打开多个租车平台 | P0 |
| **价格对比表** | 手动输入价格，自动对比计算 | P0 |
| **价格历史** | 记录价格变化趋势 | P1 |
| **数据可视化** | 品牌、能源类型分布图表 | P1 |
| **价格监控** | 关注车型，价格变动提醒 | P2 |

| 指标 | 目标 |
|------|------|
| **用户完成一次比价的时间** | < 5分钟 |
| **价格对比准确率** | 100%（用户手动输入） |
| **页面加载速度** | < 3秒 |

---

## 9. 局限性说明

| 局限 | 原因 | 解决方案 |
|------|------|----------|
| 无法自动抓取真实价格 | 各平台有反爬虫机制 | 用户手动输入价格 |
| 需要VPN访问 | 部署在国外平台 | 可迁移至国内服务器 |
| 价格监控依赖浏览器 | 无服务器端定时任务 | 可升级为服务器端监控 |
